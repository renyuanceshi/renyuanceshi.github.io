# How to generate UUID

## Procedure
1. [Windows] Press "Win + R", input cmd and run:  Powershell -NoExit -Command "[guid]::NewGuid()"
2. [v2rayN] Start v2rayN, select "Servers"=>"Add[VLESS]server"=>"Generate"
